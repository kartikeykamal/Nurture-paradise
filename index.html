Connect your Database

Nurture Paradise uses Firebase to save your account, garden, logs, cart and diagnosis history. Setup takes 2 minutes and is free forever.

1Go to console.firebase.google.com → Create a project (any name)
2In the project: Authentication → Get started → Enable Email/Password
3Firestore Database → Create database → Start in test mode
4Project Settings (⚙️) → Your apps → Add Web app → Copy the firebaseConfig object and paste it below

Paste the config object (with or without the surrounding const firebaseConfig =). It's saved locally on your device only.
